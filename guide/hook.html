    # 基本参数

    请不要在除Markdown变量以外的变量中使用html标签 这可能会导致意想不到的错误

    规范的参数如下 (打*为必须有的参数)

    参数 说明
    *api_name 插件的名字
    *api_profile 插件的简介
    *version 插件的版本
    *author 插件的作者
    *api_address 插件的请求地址
    *request_par 插件的请求参数及解释
    *return_par 插件的返回参数及解释
    type 插件的分类(无此项自动归为一些工具分类)

    支持MarkDown的项

    • api_profile
    • api_address
    • request_par
    • return_par

    基本格式

    See More

     























    <?php
include_once($_SERVER['DOCUMENT_ROOT'].'/services/until.php');

$api_name = 'x';
$api_profile='x';
$version='1.0';
$author='x';
$type='x';
$api_address="
|请求方式|请求地址|说明|
|---|---|---
|get|/api/插件所在文件夹名字|-|";
$request_par='
|参数|说明|
|---|---|
|*`x`|-|';
$return_par='
|基础数据|说明|
|---|---|
|`/`|-|';
if ($_SERVER['REQUEST_URI'] != '/api/') {
    ...你的api代码...
}
?>

    # 公用函数

    # _return_

    function _return_($context,$status=200,$location=false)

    函数自动配置跨域请求头

    参数

    _return_(内容, 状态码[默认200],是否重定向到内容中的URL[默认false，不适用于数组])

    方法1

     




    include_once($_SERVER['DOCUMENT_ROOT'].'/services/until.php');
...
$context = ["example"=>114514];
_return_($context);

    输出1

    {
    "status": 200,
    "data": {
        "example": 114514
    },
    "time": 1680689829
}

    方法2

    include_once($_SERVER['DOCUMENT_ROOT'].'/services/until.php');
...
_return_("https://cdn.imlazy.ink:233/img/background/95468130_p0.jpg",114514);

    输出2

    {
    "status": 114514,
    "data": "https:\/\/cdn.imlazy.ink:233\/img\/background\/95468130_p0.jpg",
    "time": 1680691622
}

    方法3

    include_once($_SERVER['DOCUMENT_ROOT'].'/services/until.php');
...
_return_("https://cdn.imlazy.ink:233/img/background/95468130_p0.jpg",200,true);

    输出3 网页被重定向到https://cdn.imlazy.ink:233/img/background/95468130_p0.jpg

    图片预览:

    # curl_get

    function curl_get($url,$data=[])

    注意: 网页内容必须为json，函数返回数据类型为数组

    其实不是json也可以，会直接返回网页内容(不建议)

    参数

    curl_get(需要GET的网址, GET参数[默认为空])

    # 示例插件

    <?php
include_once($_SERVER['DOCUMENT_ROOT'].'/services/until.php');

$api_name = '测试插件';
$api_profile='只是一个测试插件';
$version='1.0';
$author='molanp';
$type='野兽先辈';
$api_address="
|请求方式|请求地址|说明|
|---|---|---
|get|[/api/test](/api/test)|哼哼哼|";
$request_par='
|参数|说明|
|---|---|
|*`homo`|一个随机恶臭数字|';
$return_par='
|基础数据|说明|
|---|---|
|`level`|恶臭的等级|';
if ($_SERVER['REQUEST_URI'] != '/api/') {
    if (isset($_GET['homo'])) {
        $homo = $_GET['homo'];
    } else {
        $homo= NULL;
    }
    if (!empty($homo)) {
        $data = ["level"=>$homo*114514];
        _return_($data, 200);
    } else {
        _return_("请求格式有误", 400);
    }
}
?>

    请求

    http://example.com/api/test?homo=114514 返回

    {
    "status": 200,
    "data": {
        "level": 13113456196
    },
    "time": 1680694386
}

    http://example.com/api/test或http://example.com/api/test?ysxb=114514等 返回

    {
    "status": 400,
    "data": "请求格式有误",
    "time": 1680694452
}
