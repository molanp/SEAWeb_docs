      # 基本参数

      DANGER

      请不要在除Markdown变量以外的变量中使用html标签 这可能会导致意想不到的错误

      规范的参数如下 (打*为必须有的参数)

      参数 说明
      *api_name 插件的名字
      *api_profile 插件的简介
      *version 插件的版本
      *author 插件的作者
      *api_address 插件的请求地址
      *request_par 插件的请求参数及解释
      *return_par 插件的返回参数及解释
      type 插件的分类(无此项自动归为一些工具分类)

      支持MarkDown的项

      • api_profile
      • api_address
      • request_par
      • return_par

      基本格式

      DETAILS

       























      <?php
include_once($_SERVER['DOCUMENT_ROOT'].'/services/until.php');

$api_name = 'x';

$return_par='

    ...你的api代码...
}

      目录结构
//注:插件文件夹(name_generator)将被放置在网站目录`api`文件夹内

      <?php

$api_name = '亚文化取名机';
$api_profile='一键生成最潮的亚文化网名';
$version='1.0';

$type='一些工具';
$api_address=re_add(['GET'],['/api/name_generator'],['-']);
$request_par=re_par(['None'],['无需任何参数']);
$return_par=re_par(['data'],['你的亚文化名字']);

$el1 = ["废墟",..., "无聊"];

$el3 = ["天使", ...,"糖"];

if (handle_check()) {
    _return_($el1[array_rand($el1)].$el2[array_rand($el2)].$el2[array_rand($el2)]);
}
?>
